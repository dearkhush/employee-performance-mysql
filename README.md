# Employee Performance MySQL Project

This is a MySQL mini project created to practice and demonstrate core SQL concepts
using a simple employee performance dataset.

## Database Design
The database contains three tables:
- departments: stores department information
- employees: stores employee details and department mapping
- performance: stores yearly performance ratings of employees

Relationships:
- One department can have many employees
- One employee can have multiple performance records (year-wise)

## SQL Concepts Covered
- INNER JOIN and LEFT JOIN
- GROUP BY and HAVING
- Window functions (RANK)
- Views and temporary tables
- Transactions using COMMIT and ROLLBACK

## Tools Used
- MySQL
